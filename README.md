# MentalBERT vs ClinicalBERT

## Overview
Comparative analysis of MentalBERT and ClinicalBERT for Sentiment analysis of mental-health-related text.

## Models Used
- MentalBERT
- ClinicalBERT

## Technologies
- Python
- PyTorch
- HuggingFace Transformers
- Google Colab

## Dataset
Domain-specific dataset (not public). Preprocessing and training code included.

## Results
MentalBERT outperforms ClinicalBERT in accuracy, precision, recall, and F1-score.

## Paper Status
Conference paper under review.

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Open `notebooks/experiments.ipynb`
3. Run all cells
